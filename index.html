Générateur de signaux PWM
PWM signal generator

Objectifs :

L'objectif de ce projet est de concevoir un prototype expérimental pour un laboratoire de recherche, capable de générer quatre signaux numériques synchronisés sur des sorties GPIO. Ce système doit permettre de tester des cartes électroniques et de valider des protocoles de communication avec les spécifications suivantes :

S1 (Référence) : Rapport cyclique variable (1% à 99%), fréquence réglable > 1 kHz.
S2 : Identique à S1 avec une phase réglable de 0 à 360°.
S3 : Fréquence multiple de S1 FS3 = q*FS1 avec q = 0,5n + 1 (n entier de 0 à 10).
S4 : Identique à S3 avec une phase indépendante réglable de 0 à 360°.
Contrôle : Pilotage à distance via une interface Web Wi-Fi (mode Point d'Accès).
Fiabilité : Persistance des signaux en cas de perte de connexion et jitter minimal.

Objectives:

The aim of this project is to develop an experimental prototype for a research laboratory, capable of generating four synchronized digital signals on GPIO outputs. This system will be used to test electronic boards and verify communication protocols with the following specifications:

S1 (Reference): Variable duty cycle (1% to 99%), adjustable frequency > 1 kHz.
S2: Same as S1 with adjustable phase from 0 to 360°.
S3: Frequency multiple of S1 FS3 = q*FS1 where q = 0.5n + 1 (n integer from 0 to 10).
S4: Same as S3 with independent adjustable phase from 0 to 360°.
Control: Remote management via a Wi-Fi Web interface (Access Point mode).
Reliability: Signals must persist during Wi-Fi loss with minimal jitter.
